# 🤖 InterviewFilterer AI
### Autonomous AI Recruitment Agent | Intelligent Resume Screening & Candidate Ranking

InterviewFilterer AI is an AI-powered recruitment ecosystem that combines **Natural Language Processing (NLP)** with **Intelligent Candidate Analytics**. It automatically analyzes resumes, extracts candidate skills and experience, evaluates job compatibility, and generates ranked shortlists in real time—all without manual screening.

## 📖 Overview

Traditional recruitment processes are often slowed by manual resume reviews, inconsistent evaluations, and large applicant volumes. **InterviewFilterer AI** solves this challenge by leveraging advanced AI models to automatically parse resumes, identify relevant skills and qualifications, and compare candidates against job requirements.

The platform provides recruiters with intelligent candidate rankings, transparent evaluation metrics, and actionable hiring insights, enabling faster, fairer, and more data-driven recruitment decisions.

---

## ✨ Key Features

### 📄 AI Resume Parsing
Automatically extracts candidate information including skills, education, certifications, projects, and work experience.

### 🎯 Intelligent Candidate Matching
Matches resumes against job descriptions using NLP-based semantic analysis and generates compatibility scores.

### 📊 Automated Candidate Ranking
Ranks applicants based on skill relevance, experience level, and overall job fit.

### 🔍 Smart Filtering Engine
Allows recruiters to filter candidates based on custom hiring criteria and keyword requirements.

### 💡 Explainable AI Recommendations
Provides transparent reasoning behind candidate rankings and selection decisions.

### ⚡ Accelerated Hiring Workflow
Reduces manual screening efforts and significantly improves recruiter productivity.

## 🔄 Workflow

### 1. Upload Resume
Candidates submit resumes through the platform.

The AI engine parses resumes and extracts relevant candidate information.

### 3. Analyze Job Fit
The system compares candidate profiles against job descriptions using semantic matching.

### 4. Calculate Compatibility Score
A candidate score is generated based on skills, qualifications, experience, and relevance.

Candidates are automatically ranked according to their suitability for the role.

Recruiters review top-ranked candidates and proceed with interviews.

## 🚀 How to Deploy on Vercel

The frontend application is built with Next.js and is fully optimized for deployment on Vercel.

### 1. Clone the Repository

```bash
git clone https://github.com/sroshans025/interviewfilterer.git
cd interviewfilterer
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

```env
NEXT_PUBLIC_API_URL=
DATABASE_URL=
OPENAI_API_KEY=
```

### 4. Push to GitHub

Commit and push your code to GitHub.

### 5. Deploy to Vercel

1. Open the Vercel Dashboard
2. Click **Add New → Project**
3. Import the InterviewFilterer repository
4. Configure environment variables
5. Click **Deploy**

Your AI-powered recruitment assistant will be live within minutes.
